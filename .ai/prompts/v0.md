# V0 Component Generator

## Instructions
1. Analyze the required component(s) carefully.
2. Generate a detailed prompt for each component.
3. Make sure the prompt mentions to keep the style simple black and white "Vercel style" using shadcn/ui
4. Please note that shadcn is now installed using `npx shadcn@latest init` and `npx shadcn@latest add button`
5. Create a <code> block for the link so I can copy it easily.
6. Ensure the prompt is URL-encoded in the link
7. Remove all line breaks from the prompt

## Component Link Format
`https://v0.dev/chat?q={url_encoded_prompt}`

## Example
`https://v0.dev/chat?q=Create%20a%20responsive%20navbar%20with%20a%20logo%2C%20navigation%20links%2C%20and%20a%20mobile%20menu.%20Use%20Tailwind%20CSS%20for%20styling.`
